# ALVR Companion

![](/assets/gtk-app.png)

A small tool for managing ALVR from Linux distribution repositories.

Main features
- APK Download Management and Installation
- ADB Device Monitoring
- USB Forwarding Configuration
- Interactive UI

# Install 
`pip install -r requirements.txt`
